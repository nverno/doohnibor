LittleJohn is a trading app used to track, discover, and trade stocks, 
aiming to be a pixel-perfect clone of the popular trading app,
[[https://robinhood.com/][RobinHood]]. It is built on Ruby on Rails, PostgresSQL,
and React/Redux.
